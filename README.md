# ✨ StyleBot — AI Fashion Stylist

An AI-powered fashion stylist chatbot that gives complete head-to-toe 
outfit recommendations for Indian occasions.

## 🌐 Live Demo
https://stylebot-three.vercel.app

## ✨ Features
- Complete look recommendations (outfit, jewellery, makeup, hairstyle, footwear)
- Personalized by city, skin tone, skin type and body type
- Women, Men and Kids styling
- Budget slider with real-time sync
- User login and style history
- Share looks via WhatsApp/Instagram

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- AI: Groq API (LLaMA 3.3)
- Database: Supabase (PostgreSQL)
- Backend: Vercel Serverless Functions
- Auth: Supabase Auth
